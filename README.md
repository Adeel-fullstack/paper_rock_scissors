# ✊✋✌️ Rock Paper Scissors Game

A simple and interactive **Rock Paper Scissors game** built using **HTML, CSS, and JavaScript**.  
This project demonstrates core front-end development skills including **DOM manipulation, event handling, and basic game logic**.

---

## 🌐 Live Demo
👉 https://Adeel-fullstack.github.io/paper-rock-scissors/

---

## 🚀 Features
- User vs Computer gameplay  
- Random computer move generation  
- Instant result display (Win / Lose / Draw)  
- Interactive and responsive UI  
- Real-time updates using JavaScript DOM manipulation  

---

## 🛠️ Tech Stack
- HTML5  
- CSS3  
- JavaScript
